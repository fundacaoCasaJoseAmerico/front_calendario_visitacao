# React + TypeScript + Vite

Este é um modelo de projeto que junta o necessário para começar rápido:

- React: Para construir as telas e componentes do site.

- TypeScript: Para ajudar a escrever um código mais seguro, que avisa sobre bugs
  antes de eles acontecerem.

- Vite: A ferramenta que faz tudo "rodar" (compilar) de forma absurdamente
  rápida. É ele que dá aquela atualização instantânea no navegador.

- shadcn/ui: Biblioteca de componentes prontos e bonitos, que seguem as melhores
  práticas de design.
